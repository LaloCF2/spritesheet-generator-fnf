¡Bienvenido a Spritesheet Generator FNF!

La herramienta definitiva creada en la web para optimizar y crear spritesheets listos para Friday Night Funkin'. Esto es lo que puedes hacer aquí:

  • Escanear de Cero: Sube un PNG en la web y fusionará, recortará las cajas de tus dibujos automáticamente sin dejar fragmentos sin usar.
  • Optimizador: Sube tu XML y tu PNG. El sistema se encargara de darte una imagen mas optimizada con su xml modificado, reduciendo la resolucion de tu personaje.
  • Laboratorio de Offsets: Ajusta posiciones, escala, duplica frames, renombra en masa y visualiza cómo lucirá cada animación.
  • Psych Live Editor: Crea el archivo JSON de tu personaje alineando posiciones, barra de vida y la cámara en tiempo real.
  • Compresor PNG Pro: Sube tu imagen PNG y la herramienta bajará su peso drásticamente conservando su resolución.

¡Novedades en la Versión 1.1! ✨

El proyecto se ha actualizado. Aquí tienes los últimos cambios implementados v1.1:

  • [NUEVO] Compresor PNG: Ahora puedes optimizar el peso en MB de tus PNG directamente en el navegador, sin usar páginas externas.
  • [NUEVO] Compresión ZIP Inteligente: Al optimizar, se descargará un archivo .ZIP conteniendo tu nuevo PNG y el XML listo para usarse.
  • [NUEVO] Psych Onion Skinning: Activando el Efecto Fantasma podrás ver el cuadro anterior translúcido para alinear las animaciones perfectas.
  • [NUEVO] Temas: Ahora puedes seleccionar un tema al gusto para mejorar la estetica de la web.
