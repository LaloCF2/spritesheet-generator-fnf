¡Bienvenido a Spritesheet Generator FNF

La herramienta definitiva creada en la web para optimizar y crear spritesheets listos para Friday Night Funkin'. Esto es lo que puedes hacer aquí:

  • Escanear de Cero: Sube un PNG y la web detectará, fusionará y recortará las cajas de tus dibujos automáticamente sin batallar.
  • Optimizador: Sube tu XML y tu PNG del personaje para poder editar sprites o frames mal acomodados sin necesidad de editar el xml a mano.
  • Psych Character Buldier: Puedes crear un archivo JSON sin necesidad de entrar al Character Editor del juego, todo se muestra en tiempo real.
